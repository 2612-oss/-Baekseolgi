클린 팬덤 피드백 플랫폼
====
# 기획 배경 및 문제점
악플과 비판의 혼재: 현재 SNS와 커뮤니티에는 원색적인 비난(악플)과 팬들의 애정 어린 건의 사항(건설적 비판)이 뒤섞여 있어, 아티스트의 멘탈을 훼손하는 심각한 사이버 불링 문제가 발생하고 있습니다.
공식적인 소통 창구의 부재: 팬들은 코디, 안무, 콘셉트 등에 대해 아쉬운 점이나 좋은 아이디어가 있어도 이를 기획사나 아티스트에게 '정제된 형태'로 전달할 공식적인 창구가 없습니다.

# 기획 배경 및 문제점
AI 문맥 분석 필터링: 단순 금지어 차단이 아닌, 문맥을 파악하는 AI를 도입합니다. 감정적이고 공격적인 언어는 필터링하고, "옷이 이게 뭐냐"라는 비난을 "의상 스타일링의 개선이 필요하다"는 객관적인 피드백으로 순화/추출합니다.
데이터 리포트 제공: 정제된 피드백을 모아 카테고리별(음악, 스타일링, 자체 콘텐츠 등) 통계 데이터로 시각화합니다. 아티스트 개인이 아닌 소속사(A&R, 마케팅팀)에 리포트 형식으로 전달하여 실무에 즉시 반영할 수 있게 돕습니다.

# 기대 효과
팬은 자신이 좋아하던 아이돌의 부족한 점이 바뀔수 있어서 좋고 아이돌은 무분별한 악플로부터 보호 받고 팬들의 피드백을 수용할수 있어서 좋습니다. 또한 기획사는 객관적인 팬들의 여론을 수집하여 부족한 부분에 더 지원해 리스크를 관리하고 경제적인 이득을 볼수있습니다.
